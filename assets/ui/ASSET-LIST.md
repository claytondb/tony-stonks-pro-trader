## Asset Production Notes

### File Naming Convention
```
[category]_[name]_[variant]_[state].[ext]
Examples:
- btn_primary_normal.png
- btn_primary_hover.png
- icon_play_24.svg
- portrait_tony_suit.png
```

### Export Settings
- PNG: 32-bit with alpha
- SVG: Optimized, viewBox preserved
- Sprite Sheets: Power-of-2 dimensions, JSON data

### 9-Slice Guides
Include slice guides in separate JSON file for each 9-slice asset:
```json
{
  "top": 16,
  "right": 16,
  "bottom": 16,
  "left": 16
}
```
